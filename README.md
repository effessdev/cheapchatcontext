# CheapChatContext

## Usage

Download and run the installer to install the app.

Now, when you run the command `ccc` from the root of your project directory, the app copies

- the project structure as an ASCII tree (excluding files according to `.gitignore`)
- AGENTS.md (if present)
- other files that you may add in `ccc.config.json`

into your clipboard; properly formatted and ready to type your prompt right away after Ctrl + V.

[Read the docs](./docs/README.md)
